Planteo del problema: Instalación de los motores de base de datos, utilizando Docker Container:   
 
• Investigar  cómo  montar  un  servidor  de  HBase  usando  un  container  de  Docker  y  usarlo  
desde Python con alguna librería disponible.  
• Investigar cómo montar un servidor de MongoDB usando un container de Docker y usarlo 
desde Python con alguna librería disponible.  
• Investigar cómo montar un servidor de Redis usando un container de Docker y usarlo desde 
Python con alguna librería disponible.  
• Preferiblemente genere un documento de Docker Compose para la instalación de toda la 
arquitectura necesaria.  
• Para que su respuesta sea lo más objetiva posible, desarrollé el ejemplo en Windows como 
en Linux, este último preferiblemente en modo consola, sin interfaces gráfica instalada.   
Carga de datos: 
Crear  un  script  de  Python  que  cargue  el  dataset  eCommerce  Purchase  History  from 
Electronics  Store,  el  cual  cuenta  con  2  millones  de  registros,    en  los  3  motores  de  datos 
instalados en el paso anterior, siguiendo las reglas y esquema de cada uno de ellos.  Recuerde  optimizar  los  procesos  para  que  el  tiempo  invertido  en  la  migración  sea  lo  mas  
corto posible.   Enlace de descarga:  eCommerce purchase history from electronics store   
Consultas nativas: 
Al  igual  como  usted  realiza  consultas  de  tipo  SQL,  por  ejemplo,  en  PostgreSQL  o  MySQL,  
utilizando herramientas como pgAdmin o Workbench, solo por mencionar algunos, 
investigue como desarrollar consultas nativas sobre los motores de datos solicitados para 
este trabajo, de forma que pueda indicar: 
• Herramienta por utilizar para conectarse y aplicar la consulta 
• ¿Cómo se escribe la consulta? 
• ¿Cómo se visualiza el resultado obtenido?   
Consultas desde Python: 
Crear  un  script  (uno  por  cada  base  de  datos)  en  Python,  que  permita  desarrollar  las 
siguientes consultas en las bases de datos, (debe investigar cómo hacer las consultas y 
cada consulta debe estar registrada en la documentación):  
• ¿Cuál es la categoría más vendida?  
• ¿Cuál marca (brand) generó más ingresos brutos?  
• ¿Qué mes tuvo más ventas? (use formato UTC)  
 Experimente y analice los tiempos de respuesta de cada consulta en cada base de datos.
